Onde colocar o código JS?

Pode ser executado de duas formas. Com a tag SCRIPT ou em um arquivo separado! O aqruivo script externo tem que ser posto no lugar certo do código HTML, pois afeta o carregamento da página! Pode ser que ao tentar manipular o H1, se o script no HEADER, o código não funcionará, pois isso faz com que o JS carregue antes da tag.
